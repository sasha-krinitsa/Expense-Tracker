# Expense Tracker

Автор: Александра Криницына (sasha_krinitsa)

## Краткое описание программы

Программа "Трекер расходов" позволяет записывать, сохранять и анализировать свои расходы. Создана как небольшой проект для аттестации по курсу.

Требования: Python 3.x

## Как запустить приложение


Вам нужно загрузить файлы, а потом запустить main.py. Программа управляется с помощью консольных команд.

Обычно в качестве команд используются цифры от 1 до 8, но в случаях, когда требуется подтверждение пользователя, программа запрашивает сообщение "да". 

Не беспокойтесь о регистре букв и не слишком беспокойтесь о пробелах, пока они не в середине строки, но будьте внимательны к раскладке клавиатуры.

В случае запроса подтверждения любая команда кроме "да" будет засчитана за "нет".

Имя файла для сохранения истории — "history.json". Это можно изменить, изменив имя файла в начала файла main.py.


## Примеры использования


Доступные функции:

    1. Добавление записей с указанием суммы, категории и даты

    2. Просмотр сохранённых записей и фильтр по датам и категориям

    3. Поиск по категориям и датам: как за один день, так и за период

    4. Рассчёт суммы расходов за выбранный период

    5. Удаление записей

    6. Чтение и сохрание записей в отдельный файл формата json


Благодаря широкому функционалу трекера пользователь может отследить и проанилизовать свои расходы. Это поможет сократить их в будущем.
